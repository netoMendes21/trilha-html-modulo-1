Tags h1 até h6

H1: É o título principal da página, representando o tema geral da página. Deve ser único em cada página.

H2: Subtítulos que dividem a página em seções principais.

H3: Subtítulos que detalham as seções criadas pelos H2.

H4, H5, H6: São usados para subdividir ainda mais as seções, proporcionando uma hierarquia mais detalhada.

Tags para texto

Tag p utilizamos para criar parágrafo.

Tag mark utilizamos para destacar textos. Exemplo: texto marcado utilizando a tag mark.

Tag u é utilizada para deixar o texto escrito com um "underline". Exemplo: texto utilizando a tag u

Tag small representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada. Exemplo: texto utilizando a tag small

Tag span é um contêiner genérico em linha para conteúdo fraseado , que não representa nada por natureza.

Tag sub Especifica o texto embutido que deve ser exibido como subscrito por motivos exclusivamente tipográficos. Exemplo: texto utilizando a tag sub

Tag sup Especifica o texto embutido que deve ser exibido como sobrescrito por motivos exclusivamente tipográficos. Exemplo: texto utilizando a tag sub

Tags para criar listas

Tag ul para criar listas não ordenadas.

Exemplo:
  • Exemplo 1
  • Exemplo 2
  • Exemplo 3

Tag ol para criar listas ordenadas.

Exemplo:
  1. Primeiro Exemplo
  2. Secundo Exemplo
  3. Terceiro Exemplo

Tag li representa cada item de uma lista, tanto na ul(lista não ordenada),como na ol(lista ordenada);

Exemplo:
  • Item 1
  • Item 2
  • Item 3

HTML Semântico

Tag header representa um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho, mas também outros elementos como um logotipo, rótulos de cabeçalho, formulário de pesquisa, e outros.

Tag main representa o assunto principal da página, podendo apenas existir uma única por página.

Tag footer representa o rodapé da página.

Tag article representa a criação de artigos na página, para colocar assuntos independentes ou relacionados com o assunto principal.

Tag strong é muito parecida com a tag b(bold), porém com uma "força" semântica maior.
